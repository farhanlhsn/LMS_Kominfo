Implement course/lesson discussions, comments, votes, accepted answer, announcements, live sessions via provider adapter/custom link, attendance, notifications, calendar events.

## Required reading

- Keep the app buildable.
- Respect multi-tenancy and RBAC.

- TypeScript passes.
- Backend and frontend build.
- Migrations run.
- Critical tests pass.
- APIs follow `/api/v1` and standard response format.
- UI is usable and responsive.
- Tenant isolation is enforced.
- Audit logs are created for sensitive operations.
