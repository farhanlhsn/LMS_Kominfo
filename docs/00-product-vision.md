# Product Vision

Build a general AI-powered LMS similar to Coursera + Moodle.

The platform must support self-paced learning, cohort-based learning, instructor-led live sessions, enterprise organizations, marketplace course sales, certificates, AI tutoring, and plugin-based learning activities.

## Core user roles

- Super Admin
- Organization Admin
- Course Manager
- Instructor
- Assistant Instructor
- Reviewer / Assessor
- Mentor / Coach
- Learner
- Content Creator
- Finance Admin
- Support Admin
- Guest

## Complete feature scope

### Learning

- Course catalog
- Course detail page
- Course builder
- Module, lesson, activity structure
- Learning paths
- Cohorts/classes/batches
- Scheduled course delivery
- Video, article, PDF, file, link, audio
- SCORM, xAPI, H5P-style interactive content
- 3D content and interactive simulation plugin
- Code runner / coding exercise plugin
- Advanced learning workspace
- Side-by-side panels
- Pop-out dual-monitor support
- Transcript sync
- Notes and timestamp notes
- Bookmarks
- AI panel while learning

### Assessment

- Quiz engine
- Question bank
- Advanced question types
- Assignment and project submission
- Rubric grading
- Peer review
- Group assignment
- Code exercise grading
- Proctoring-ready architecture
- Plagiarism checker abstraction

### Engagement

- Goals
- Streaks
- XP
